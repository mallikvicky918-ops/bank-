Functional Requirement
•	FR-PAY-01: The system shall allow users to add, update, and delete their bank account details securely.
•	FR-PAY-02: The system shall validate bank account details (account number, IFSC/SWIFT code) before saving them.
•	FR-PAY-03: The system shall process payments only after successful verification of the user’s bank details.
Security Requirement
•	SR-PAY-01: The system shall encrypt all bank account details both in transit and at rest.
•	SR-PAY-02: The system shall restrict access to bank details to authorized users and system components only.
Non-Functional Requirement
•	NFR-PAY-01: The system shall complete payment transactions within 5 seconds under normal operating conditions.
